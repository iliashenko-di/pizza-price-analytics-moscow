Выберите город Papa Johns и ресторан Dodo. Сбор запускается через GitHub Action: Playwright обновит snapshot, а эта страница подтянет новые данные после публикации Pages.

После запуска GitHub Action установит Chromium, запустит Playwright-сборщик, проверит качество данных, закоммитит `data/pizza-snapshot.*` и обновит GitHub Pages.

Сравнение пицц

Выберите пиццу Papa Johns и пиццу Dodo Pizza, чтобы увидеть совпавшие варианты по размеру и тесту, а также варианты без пары.

Совпадением считается одинаковый размер и одинаковый вид теста. Варианты без точной пары выводятся отдельно.
