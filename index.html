Новый сбор данных

Выберите город Papa Johns и ресторан Dodo или включите среднюю Dodo-цену по всем московским ресторанам. Сбор запускается через GitHub Action: Playwright обновит snapshot, а эта страница подтянет новые данные после публикации Pages.

Режим Dodo
Только пиццы Стандартный борт Без половинок Без конструкторов Базовые цены
После запуска GitHub Action установит Chromium, запустит Playwright-сборщик, проверит качество данных, закоммитит `data/pizza-snapshot.*` и обновит GitHub Pages.
Открыть GitHub Action
Команду можно выполнить в терминале, где авторизован GitHub CLI.

Проверка Dodo по сегменту

Выберите пиццу Dodo Pizza, чтобы увидеть, в какой Papa-сегмент она попадает по ценам и где Dodo выше или ниже базовой цены Papa.

Сегмент Dodo определяется по ближайшей ценовой лестнице Papa: 25 см Dodo сравнивается с 23 см Papa, 30 см с 30 см, 35 см с 35 см. Размер 20 см показывается отдельно и не влияет на сегмент.

Сравнение по сегментам

Процент считается от цены Papa: `(Dodo - Papa) / Papa`. Если Dodo выше Papa, процент зеленый; если Dodo ниже Papa, процент красный.
